# 🎮 坦克大战 - 微信可直接访问的小游戏

一个完全使用 HTML5 Canvas 开发的炫酷坦克大战游戏，完美适配移动端，可直接在微信浏览器中运行。

## ✨ 游戏特色

### 🎯 核心功能
- **玩家控制**: 支持方向键/WASD键盘控制 + 移动端触摸控制
- **敌方AI**: 智能敌方坦克，会追踪、射击、躲避
- **地形系统**: 多种地形（砖块、钢铁、水域、树林、冰面）
- **道具系统**: 5种强化道具（强化、炸弹、护盾、冻结、生命）
- **关卡递进**: 无限关卡，每关难度递增

### 🎨 视觉效果
- ✨ 高质量粒子系统（爆炸、烟雾、尾焰）
- 🔥 动态爆炸动画
- 💫 炮弹轨迹拖尾效果
- 🌟 实时动态光影
- 🌊 水面波纹动画
- ⭐ 道具悬浮效果
- 🎮 流畅的坦克移动动画

### 📱 移动端优化
- 响应式设计，适配所有屏幕
- 虚拟方向键 + 射击按钮
- 触摸事件优化
- 60fps 流畅运行

### 💾 数据统计
- 实时分数显示
- 关卡进度追踪
- 本地最高分保存
- 生命值管理

## 🚀 快速开始

### 方法 1: 直接打开
```bash
# 双击 tank-battle.html 文件即可在浏览器中打开
```

### 方法 2: 使用本地服务器
```bash
# Python 3
python -m http.server 8080
# 然后访问 http://localhost:8080/tank-battle.html

# 或使用提供的脚本
python start_server.py
```

### 方法 3: 一键部署
```bash
# 使用部署向导
python deploy.py
```

## 🎮 操作说明

### 电脑端
- **WASD** 或 **方向键**: 移动坦克
- **空格键**: 发射炮弹

### 手机端
- **左侧虚拟摇杆**: 控制移动方向
- **右侧红色按钮**: 发射炮弹

## 🏗️ 技术架构

```
tank-battle.html
├── HTML5 Canvas 渲染
├── CSS3 动画和样式
└── JavaScript
    ├── 游戏引擎（60fps 渲染循环）
    ├── 粒子系统
    ├── 碰撞检测
    ├── 敌方AI
    ├── 道具系统
    └── 响应式适配
```

### 核心技术
- **渲染**: HTML5 Canvas 2D
- **动画**: requestAnimationFrame
- **输入**: 键盘事件 + 触摸事件
- **存储**: localStorage
- **网络**: 无需后端，纯前端

## 🎯 游戏规则

1. **目标**: 消灭所有敌方坦克
2. **生命**: 初始3条命，最多5条
3. **得分**: 击杀敌人获得 100×关卡 分
4. **过关**: 清空所有敌人进入下一关
5. **失败**: 生命归零游戏结束

## 🎁 道具效果

- ⭐ **强化**: 提升炮弹威力和射速
- 💣 **炸弹**: 清除屏幕所有敌人
- 🛡️ **护盾**: 无敌保护60秒
- ⏰ **冻结**: 冻结所有敌人10秒
- ❤️ **生命**: 增加一条生命

## 🐛 问题排查

**游戏无法加载?**
- 检查浏览器控制台错误信息
- 确认使用 HTTPS 访问
- 清除浏览器缓存重试

**触摸控制不响应?**
- 确保触摸的是虚拟按键区域
- 检查设备触摸功能是否正常

**微信中无法全屏?**
- 点击微信右上角"..."
- 选择"在浏览器中打开"
